# Dev log — 2026-06-08

Everything in this project was built in one day. This log captures the arc and
the reasoning, in order. For the durable architecture reference, see
[`CLAUDE.md`](../CLAUDE.md).

## Starting point
A single prototype file (`pokemon-pack-simulator-2.html`, kept at repo root for
reference) — a neon retro booster-pack opener from a Claude-on-web session.
Strong visual identity; everything in one `<script>`; no persistence, no build,
small hardcoded card pool, live PokéAPI sprites + `pokemontcg.io` price lookups.

## Phase 1 — Modular rebuild (v2)
Rebuilt the single file into a real project under hard constraints from the user:
**Bun, not npm; no third-party packages** (recent npm supply-chain compromises —
vendor/cherry-pick snippets instead). Result: vanilla JS + native ES modules,
zero dependencies, a ~30-line `Bun.serve` static dev server (`server.js`).
Split into `src/{data,state,game,services,ui}` + `styles/styles.css`. Central
pub/sub store with `localStorage` persistence. Initially used invented sets +
PokéAPI sprites + name-search pricing.

- Installed Bun (`~/.bun/bin/bun`, v1.3.14) via the official script (not npm).
- Bug: `server.js` 404'd everything because the `é` in `pokénet` made
  `import.meta.url`'s path percent-encoded → switched to `import.meta.dir`.

## Phase 2 — Realism rebuild (the big pivot)
Root-caused the "$24 Bulbasaur" bug: we showed a generic species and
name-searched `pokemontcg.io` for *some* matching card. Fix: **a card is a
specific real printing**. Rewrote around real set data:
- `services/cards.js` — `loadSet(apiSetId)` fetches a whole set in one request,
  normalizes (real image + real per-variant market price + derived tier), caches
  in memory + `localStorage`. `rarityToTier()` maps the many real rarity strings
  → our six tiers.
- Real card **images** and **prices**; deleted `data/pokemon.js` + `sprites.js`.
- **Free Prismatic Evolutions pack** gated by a cooldown (the on-ramp), paid
  sets cost currency.

## Phase 3 — Economy + UX iterations (many small requests)
In rough order:
- **Currency: shards → money** (plain dollars, same unit as card prices).
- **Sell any card**, not just duplicates.
- **Selling takes time** — listings sell over a short per-card duration…
- …then **one at a time** (a FIFO sale queue; only the front ticks).
- **Bulk sell** "all commons & uncommons" (first instant, then changed to queue).
- **Lock cards** to prevent accidental selling (binder modal toggle).
- **Achievement toasts deferred** until all cards in a pack are flipped (no spoilers).
- Fixed the reveal **price text rendering mirrored** (counter-rotate the tag, since the card flips `rotateY(180deg)`).
- **Pricing model** iterations: stopped showing the pricey reverse-holo variant
  for base cards → then made card price the **average of variant market prices**
  (per user) → finally **pack price = real sealed-pack market price** (hardcoded
  per set), card prices stay real/averaged.
- **PE cooldown** 60s → 2min.

## Phase 4 — Real pack wrapper art
- Convention: each pack loads `assets/packs/<set.id>.png`; missing → falls back
  to the set logo + holo gradient.
- Sourced real wrapper photos from the TCGplayer product-image CDN
  (`product-images.tcgplayer.com/<productId>.jpg`) and retailer `og:image`
  (Shopify); trimmed white margins with Python/PIL; downscaled with `sips`.

## Phase 5 — Structure: subtabs
- **Binder → Cards / Sealed** subtabs. Added **collectible sealed packs**:
  "Keep Sealed" in the buy view banks a pack unopened; open it later from the
  Sealed tab.
- **Shop → Buy / Sell** subtabs; folded the standalone "Open" top tab into
  **Shop ▸ Buy**. Top nav is now **Shop / Binder**.

## Phase 6 — Content (packs added through the day)
Started with Prismatic Evolutions, 151, Surging Sparks, Paldean Fates, Fossil.
Added: **Evolving Skies** (Rayquaza wrapper), **Obsidian Flames** (Dragonite
wrapper — PE has no Dragonite art, it's Eeveelutions), **Base Set** + **Crown
Zenith**, **EX Dragon** + **Ascended Heroes**. **11 packs total.**
- `rarityToTier` fixes along the way: vintage "Rare Holo" → holo; ex-era
  "Rare Holo EX" → ultra.
- **Ascended Heroes caveat**: brand-new set (Jan 2026) — `pokemontcg.io` has no
  live prices and no secret-tier rarities yet, so its card values are
  placeholders (rarity floors) until the data populates; it self-heals on the
  next cache refresh.

## Phase 7 — Hosting
- Created repo **github.com/finnstevens/pokenet** (public).
- Installed `gh` CLI to `~/.local/bin/gh` (brew was broken; downloaded the binary).
- **GitHub Pages via Actions** (`.github/workflows/deploy.yml`) — auto-deploys on
  push to `main`. Live at **https://finnstevens.github.io/pokenet/**.

## Price fact-check (done at user request)
Verified card prices are accurate real-market values (Moonbreon $2,181, Base Set
Charizard $572, PE Umbreon ex $1,558, etc.). The earlier "$46 Base Set" was the
*expected value of a pack's contents* (correct math), not the sealed-pack price —
which is why we switched pack pricing to real sealed-pack market values.
